¿Qué es Data Science?

La Ciencia de Datos es un campo interdisciplinario que utiliza métodos científicos, procesos y algoritmos para extraer conocimiento de datos estructurados y no estructurados.

"Sin datos, solo eres otra persona con una opinión."
— W. Edwards Deming

Combina estadística, matemáticas, programación y conocimiento de negocio para resolver problemas complejos.

Ciencia

Método científico aplicado al análisis.

Tecnología

Algoritmos computacionales potentes.

Negocios

Decisiones basadas en evidencia real.

Machine Learning

Automatización y predicción.

El Ecosistema de Datos

Diferentes roles para diferentes objetivos

Data Analyst

Interpreta datos pasados. Crea reportes y dashboards para responder preguntas de negocio.

Business Analyst

El puente entre IT y negocio. Traduce necesidades empresariales en requisitos técnicos.

Data Scientist

Predice el futuro. Construye modelos de ML y algoritmos avanzados.

El arquitecto. Construye pipelines y mantiene la infraestructura de datos.

SQL (Structured Query Language)

La base de todo profesional de datos

No puedes analizar datos si no puedes obtenerlos. SQL te permite comunicarte con bases de datos para filtrar, agrupar y extraer la información que necesitas.

SQL se utiliza en el 95% de los proyectos de datos.

consulta.sql
SELECT categoria, COUNT(*) as total_ventas FROM tabla_ventas WHERE fecha > '2023-01-01' GROUP BY categoria ORDER BY total_ventas DESC;

LABORATORIO PRÁCTICO

Manos a la Obra con R

Vamos a analizar el dataset mtcars incluido en R. Contiene datos de 32 modelos de automóviles (1974).

1. Explorar

Entender qué datos tenemos.

2. Visualizar

Ver patrones visualmente.

3. Calcular

Medias y desviaciones.

Paso 1: Exploración

01_exploracion.R
# Cargar el dataset (viene por defecto en R) data("mtcars") # Ver las primeras 6 filas head(mtcars) # Ver estructura (tipos de datos) str(mtcars) # Resumen estadístico rápido (min, max, media) summary(mtcars) # Dimensiones (filas x columnas) dim(mtcars) # Output: 32 11

Paso 2: Visualización (ggplot2)

02_graficos.R
library(ggplot2) # Gráfico de dispersión: Peso (wt) vs Consumo (mpg) ggplot(mtcars, aes(x = wt, y = mpg)) + geom_point(color = "#0284c7", size = 3) + labs(title = "Peso vs Eficiencia", x = "Peso (1000 lbs)", y = "Millas por Galón") + theme_minimal() # Histograma de Consumo ggplot(mtcars, aes(x = mpg)) + geom_histogram(bins = 10, fill = "#0d9488", color = "white")

Paso 3: Estadística Descriptiva

03_stats.R
# Media de consumo (mpg) media_mpg <- mean(mtcars$mpg) print(media_mpg) # Desviación estándar sd_mpg <- sd(mtcars$mpg) # Correlación entre Peso y Consumo # -1 es correlación negativa perfecta (más peso, menos mpg) correlacion <- cor(mtcars$wt, mtcars$mpg) print(correlacion) # Resultado: -0.86 (Muy fuerte)

Paso 4: Modelo Predictivo

04_modelo.R
# Regresión Lineal: Predecir mpg basado en peso (wt) modelo <- lm(mpg ~ wt, data = mtcars) # Ver coeficientes summary(modelo) # Hacer una predicción # ¿Cuánto consumiría un auto que pesa 3.5 (3500 lbs)? nuevo_auto <- data.frame(wt = 3.5) prediccion <- predict(modelo, nuevo_auto) print(prediccion)
